Python3: Mutable, Immutable... Everything Is Object!


Introduction

Python est un langage de programmation puissant et flexible...

ID et Type

En Python, chaque objet a une identité...

a = 10
print(id(a))
print(type(a))

Objets Mutables

Les objets mutables peuvent être modifiés après leur création...

lst = [1, 2, 3]
print(id(lst))
lst.append(4)
print(id(lst)) # L'ID reste le même

Objets Immuables

Les objets immuables, contrairement aux objets mutables...

a = (1, 2, 3)
print(id(a))
a += (4,)
print(id(a)) # L'ID change

Importance et Traitement par Python

La distinction entre objets mutables et immuables est cruciale...

Passage d'Arguments aux Fonctions

Lorsqu'un objet est passé à une fonction, Python passe une référence...

def update_list(lst):
lst.append(4)

def update_number(a):
a += 1

lst = [1, 2, 3]
a = 10

update_list(lst)
update_number(a)

print(lst) # [1, 2, 3, 4]
print(a) # 10

Conclusion

Comprendre la différence entre les objets mutables et immuables en Python...
